# SpringBoot API

A RESTful API built with Spring Boot for Authentication and Customer Account Management.

## Tech Stack
- Java / Spring Boot
- Spring Data JPA
- H2 Database
- Jakarta Validation
- WebClient
- Swagger / OpenAPI

## Features
- Customer registration and management
- Account creation and retrieval
- Input validation and error handling
- RESTful endpoints following best practices
- API documentation via Swagger UI

## Getting Started

### Prerequisites
- Java 17+
- Maven

### Run Locally
git clone https://github.com/LalithaaD/SpringBoot-API.git
cd SpringBoot-API
mvn spring-boot:run

### API Docs
Once running, visit: http://localhost:8080/swagger-ui.html
